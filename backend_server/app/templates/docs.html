Backend API Documentation

GET
/api/health

Health check endpoint that returns server status and current time.

Response Example:

{
    "status": "ok",
    "time": "2025-11-02T12:00:00Z"
}
GET
/api/hello?name=world

Returns a greeting message. Supports optional name parameter.

Query Parameters:

  • name (optional): Name to greet

Response Example:

{
    "message": "Hello, world!"
}
POST
/api/echo

Echoes back the JSON payload sent in the request.

Request Body Example:

{
    "message": "test",
    "data": { "key": "value" }
}

Response Example:

{
    "echo": {
        "message": "test",
        "data": { "key": "value" }
    }
}
GET
/api/add?a=5&b=3

Adds two numbers provided as query parameters.

Query Parameters:

  • a: First number
  • b: Second number

Response Example:

{
    "a": 5,
    "b": 3,
    "sum": 8
}
GET
/api/items

Lists all items in the store.

Response Example:

{
    "items": [
        {
            "id": "uuid-here",
            "name": "Item 1",
            "data": {},
            "created_at": "2025-11-02T12:00:00Z"
        }
    ]
}
POST
/api/items

Creates a new item.

Request Body Example:

{
    "name": "New Item",
    "data": {
        "description": "Optional metadata"
    }
}

Response Example:

{
    "id": "generated-uuid",
    "name": "New Item",
    "data": {
        "description": "Optional metadata"
    },
    "created_at": "2025-11-02T12:00:00Z"
}
PUT
/api/items/{item_id}

Updates an existing item by ID.

Request Body Example:

{
    "name": "Updated Name",
    "data": {
        "updated": true
    }
}

Response Example:

{
    "id": "item-id",
    "name": "Updated Name",
    "data": {
        "updated": true
    },
    "updated_at": "2025-11-02T12:00:00Z"
}
DELETE
/api/items/{item_id}

Deletes an item by ID.

Response Example:

{
    "deleted": "item-id"
}
